# TabbyTab Icons

This directory contains the source files for the TabbyTab extension icons.

## Icon Files

- `tabbytab.svg` - The full detailed SVG icon (512x512)
- `tabbytab-simple.svg` - Simplified version for 48px size
- `tabbytab-16.svg` - Simplified version for 16px size
- `icon-converter.html` - A web-based tool to convert SVGs to PNGs

## How to Update Icons

To update the TabbyTab icons:

1. Edit the SVG files as needed using an SVG editor like Inkscape or Adobe Illustrator.
2. Open the `icon-converter.html` file in a browser.
3. Click "Generate Icons" to render the icons.
4. Download each icon size using the "Download" buttons.
5. Move the downloaded icons to the `/icons` directory.
6. Rebuild the extension.

## Icon Design

The current icon design features:
- Tabby cat face (orange/yellow) with blue eyes
- Browser tab/window background
- Blue square background matching the app's color scheme
- Simplified designs for smaller sizes
